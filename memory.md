我使用 yarn 作为包管理器。

我使用 Zustand 进行状态管理。

我使用 shadcn UI 作为主要的组件库，可以使用基于 shadcn UI 的其他 UI 库，例如：Magic UI、Aaceternity、Coss UI、Cult UI。

我使用 tailwind CSS 进行样式设计。

我使用 SWR 进行数据请求和缓存管理。

可以使用lib/notification.ts中的通知功能来弹出各种类型的通知，例如成功、错误、警告和信息通知。

请求后端的API都写在了services目录下，可以参考里面的代码来编写新的API请求。所有的API请见docs/swagger.yaml。
